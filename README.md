## 概要
シャレイア語辞典に関する様々な処理を行う Web システムです。
ブラウザ上でシャレイア語辞典を検索できるフロントエンドアプリの他、様々な用途のためのバックエンド API を提供しています。
オンライン辞典アプリは[こちら](https://sakalt.github.io/Qolxarvan/)で公開しています。
